🚀 Smart Trip Planner
Safe, Eco-Aware, Cost-Optimized Urban Routing System
Presented by: Team Hack
Tech Stack: HTML, CSS, JavaScript, Leaflet.js, OpenRouteService API

📌 Project Overview
Smart Trip Planner is an intelligent urban routing web application designed to go beyond traditional navigation systems like Google Maps.

While most mapping services focus only on the shortest or fastest route, our system integrates safety, environmental awareness, cost optimization, and risk intelligence into route planning.

This project addresses major urban travel challenges by combining mapping data, pollution monitoring, cost calculations, and safety analysis into a single interactive platform.

❗ Problem Statement
Traditional navigation systems have several limitations:

❌ No real-time pollution awareness (AQI not considered)

❌ No crime or accident risk visibility

❌ No detailed cost breakdown

❌ No passenger-based fare calculation

❌ Routes prioritize speed, not safety

Travelers lack complete information for making smart, safe, and eco-friendly travel decisions.

💡 Our Solution: Smart Trip Planner
Smart Trip Planner integrates multiple real-world factors into route decision-making:

✅ Intelligent multi-route comparison

✅ Air Quality Index (AQI) display

✅ Crime & accident zone alerts

✅ Passenger-based fare calculation

✅ Environmental exposure analysis

✅ Emergency SOS feature

✅ Cost-optimized routing recommendations
